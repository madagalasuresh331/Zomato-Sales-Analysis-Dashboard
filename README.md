Zomato Sales Analysis Dashboard

Project Overview

This project is a Zomato Sales Analysis Dashboard created using Microsoft Excel.
The main goal of this project is to analyze food delivery sales data and understand business performance using KPIs, charts, and reports.

The dashboard helps to track sales, profit, ratings, delivery performance, city-wise performance, and quarter-wise growth.

---

Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
- Data Cleaning
- Data Visualization

---

Key Performance Indicators

The dashboard includes the following KPIs:

- Total Sales
- Total Profit
- Average Rating
- Average Delivery Time
- Growth Percentage
- Total Orders

---

Dashboard Analysis

The dashboard shows analysis based on:

- Sales by City
- Profit by City
- Sales by Quarter
- Profit by Quarter
- Average Rating Analysis
- Delivery Time Analysis
- Growth Analysis

---

Key Insights

- City-wise analysis helps identify which cities generated more sales and profit.
- Quarter-wise analysis helps understand sales growth over time.
- Average rating helps measure customer satisfaction.
- Average delivery time helps analyze delivery performance.
- KPI cards give a quick summary of business performance.
